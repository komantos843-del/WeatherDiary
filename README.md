Movie LibraryАвтор: Данила Каншин

Описание:Графическое приложение для ведения личной кинотеки. Позволяет добавлять фильмы с проверкой данных (название [строка] , жанр [строка] , год [целое число] , рейтинг [число от 0 до 10] ), фильтровать их по жанру и году выпуска. Данные можно сохранять и загружать в формате JSON.

Возможности:

Добавление новых фильмов с валидацией ввода.
Фильтрация по жанру и году (критерии можно комбинировать).
Явное сохранение и загрузка данных через кнопки.
Автоматическая загрузка данных из data/movies.json при запуске.
История изменений с помощью Git.

Примеры использования:

Запустите приложение.
Введите данные о фильме и нажмите «Добавить фильм».
Используйте выпадающие списки для фильтрации и кнопку «Применить фильтр».
Для сохранения текущей библиотеки нажмите «Сохранить». Для загрузки из файла — «Загрузить»
