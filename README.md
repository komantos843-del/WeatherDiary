Movie Library
Автор: Данила Каншин

Описание:Приложение для хранения и фильтрации личной кинотеки. Позволяет добавлять фильмы с полями: название, жанр, год выпуска, рейтинг. Данные сохраняются и загружаются из JSON.

Примеры использования:

Запустите movie_library.py.
Введите данные о фильме и нажмите «Добавить фильм».
Используйте фильтрацию по жанру и году.
Сохраняйте и загружайте данные через кнопки.
Все изменения отслеживаются через Git.
