# Weather Diary

**Автор:** Данила Каншин

## Описание программы

Приложение «Дневник погоды» позволяет вести учёт погодных наблюдений: добавлять записи с датой, температурой, описанием и указанием осадков. Данные сохраняются в файл JSON. Доступна фильтрация по дате и температуре.

## Требования

- Python 3.8+
- Библиотека `tkcalendar` (`pip install tkcalendar`)

## Запуск приложения

1. Клонируйте репозиторий:
   ```
   git clone https://github.com/ваш_логин/weather-diary.git
   cd weather-diary
   ```
2. Установите зависимости:
   ```
   pip install -r requirements.txt  # если есть файл requirements.txt
   ```
3. Запустите приложение:
   ```
   python weather_diary.py
   ```

## Примеры использования

- Добавьте запись о погоде за сегодня: введите дату (по умолчанию — текущая), температуру (например, 15.2), описание (например, «ясно»), отметьте осадки при необходимости и нажмите «Добавить запись».
- Для фильтрации введите интересующую дату или температуру (например, показать записи с температурой выше +10°C) и нажмите «Применить фильтр».
- Все записи сохраняются в файл `data.json` и загружаются при запуске программы.
